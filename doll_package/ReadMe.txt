●操作説明
・ゲームシーン
　A/D 			=> 視点操作
　右クリック		=> 蝋燭の使用
　SPACE			=> モニター切り替え
　マスクUI(オンマウス) 	=> マスクの装着
　時計(オンマウス) 	=> 現在時刻の表示
　操作UI(クリック) 	=> 操作説明の表示
　水晶UI(クリック) 	=> モニタービュー遷移

●ゲーム内容
　ゲームの内容は「Five Night at Freddy's 2」のオマージュ作品です。
　お面を被ることで敵の襲撃を回避することができます。
　また、状況に合わせて蝋燭や水晶(モニター)を
　駆使して敵の攻撃を回避してください。

●Tips
・お蘭は蝋燭を点灯させることで襲撃を回避することができます。
・お蘭はお面で襲撃を回避することができません。
・マーガレットは子供部屋に視点を合わせることで
　襲撃ゲージを回復させることができます。
・マーガレットはお面で襲撃を回避することができません。
・牡丹は視点を変更していると時折出現します。
・牡丹は蝋燭の使用、お面の使用で襲撃されてしまいます。
・全てのエネミーはそれぞれ独自の襲撃ルートを辿ってきます。
